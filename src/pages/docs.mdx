### What is reablocks?
reablocks is a set of beautifully designed, highly customizable, Open Source React components
based on Tailwind and Framer Motion. It combines the flexibility and ease of Tailwind CSS with the 
convienance of global component themes to create a seamless development experience.

### Why reablocks?
There are many React component libraries out there, but reablocks is unique in that it 
is built on top of Tailwind CSS and framer-motion. This allows for a high level of customization
and flexibility while maintaining a simple and easy to use API. It also allows for seamless 
animations and transitions out of the box. To learn more about why reablocks, 
check out the [Philosophy](/getting-started/philosophy) page.

### How do I get started?
To get started with reablocks, check out the [Getting Started](/getting-started/setup) page.
It will walk you through how to install reablocks and get up and running with your first component.
